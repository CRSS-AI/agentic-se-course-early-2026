# Agentic SE Course

This is a proposal of curriculum for agentic software engineering given the latest developments.

Coding agents are already exceeding human throughput for many SE tasks. As model capabilities continue to improve, agents will increasingly be the primary producers of code. The engineering challenge shifts: the bottleneck is no longer writing code but ensuring that agent-produced code is correct, tested, coherent, and merge-ready, with as little human steering as possible.

This course is structured as an inverse classroom over 5 weekly sessions plus an optional capstone:

- **Week 0** — Getting Started: installation and live demo
- **Week 1** — One Coding Agent: hands-on exercises and best practices
- **Week 2** — Adapting A Coding Agent: skills, MCP, and context engineering
- **Week 3** — Solving Hard Problems: the Research-Plan-Implement workflow
- **Week 4** — Towards Agent-First SE: harness engineering and multi-agent coordination
- **Week 5** — Capstone (optional): build a project harness-first

Each week includes pre-work (courses, readings, videos) completed independently, followed by a one-hour live session for discussion and collaborative exercises.

The full curriculum is published at: https://antonio-tresol.github.io/agentic-se-course-early-2026/
